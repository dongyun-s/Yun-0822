# 안녕하세요, 백엔드 개발자를 지향하는 신동윤입니다 👋

Spring Boot 기반의 백엔드 개발을 중심으로 공부하고 있으며,  
사용자에게 안정적인 서비스를 제공할 수 있는 서버 구조와 API 설계에 관심이 있습니다.

현재는 Java, Spring Boot, MySQL을 활용한 웹 서비스 개발 역량을 키우고 있으며,  
AWS와 Docker를 활용한 배포 및 운영 환경 구성도 함께 학습하고 있습니다.

또한 Codex, Claude Code, Cursor와 같은 AI 개발 도구를 활용하여  
코드 작성, 리팩토링, 디버깅, 학습 과정을 효율화하는 방법에도 관심을 가지고 있습니다.
